📖 SYSTEM DOCUMENTATION & DATA LOGIC

⚡ 1. MOMENTUM SCREENER

The landing cards rank performance across 1-Week, 1-Month, and 3-Month windows.

  • Data Boundary: Main screen dashboard metric metrics are calculated using end-of-day automation snapshots.
  • Directory Categorization: Stocks are segregated structurally into Large-cap, Mid-cap, and Small-cap brackets across core operational industry sectors.

📊 2. CHART DECODER (SPEEDOMETER INDICATOR)

The speedometer dial inside the workspace represents an institutional consensus calculation from the TradingView Technical Analysis Engine.

  • It collects real-time technical inputs simultaneously across 26 underlying indicators (17 Moving Averages + 9 Oscillators).
  • Each indicator logs an active operational trade vote: BUY, SELL, or NEUTRAL.
  • The directional placement of the gauge needle highlights the net math layout of these combined technical indicators.

💰 3. DIVIDEND TRACKER MONITOR

Tracks ongoing structural corporate actions for target equities list.

  • Dividend Amount: Denotes the actual distribution payout cash value declared per individual equity share.
  • Dividend Yield (%): Represents the annualized percentage ratio of dividend return compared against current price.
  • Ex-Dividend Date: Critical cutoff threshold window. Equities purchased on or past this specific calendar date will not be eligible to collect the upcoming declared payout distribution.
